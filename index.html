How does JavaScript work?
Answer: JavaScript is a High Level, Synchronous, Single-Threaded, Interpreted Programming Language.It runs on google's high-performance opensource V8 engine. All the operation in JavaScript occurs in the Execution Context. Memory & Code/ Call Stack are 2 main parts of the V8 engine. When code executes memory allocation happens in Memory & another operation happens in Call Stack or Code. Inside Call Stack we have two parts, one is Event Loop and another one is Callback Queue. All the returned data & values of the variables are stored in memory. The Callback Queue calls a single line of code at a time because this is a Single-Threaded programming language. This is how JavaScript works.
